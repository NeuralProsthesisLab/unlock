Detailed Description

This class holds an Enobio3G/StarStim sample which consists on an integer value for each channel and an extra value that informs about the channels that are reported in that sample.

It returns an integer that reports which channels are present in the current sample

Returns:
The returned value is organized at bit level. The bits set to '1' mean that the channels are being reported. The data for those channels whose channel info bits are set to zero is undefined. The least significant bit corresponds to channel 0
